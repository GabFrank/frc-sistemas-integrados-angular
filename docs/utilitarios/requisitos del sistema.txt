ESPECIFICACIONES DEL SISTEMA
===========================

1. ARQUITECTURA GENERAL
----------------------
- Tipo: Aplicación distribuida multicapa
- Patrón de arquitectura: Cliente-Servidor
- Protocolo de comunicación: GraphQL

2. COMPONENTES DEL SISTEMA
-------------------------
2.1 Base de Datos
- Motor: PostgreSQL 16
- Características principales:
  * Replicación lógica habilitada
  * Alta disponibilidad
  * Sistema de respaldo automatizado

2.2 Backend
- Framework: Spring Boot (Java)
- Características principales:
  * API GraphQL
  * Servicios RESTful
  * Gestión de autenticación y autorización
  * Manejo de transacciones
  * Logging y monitoreo

2.3 Frontend
a) Aplicación de Escritorio
- Framework: Angular 15 + Electron
- Características:
  * Interfaz de usuario con Angular Material
  * Modo offline
  * Actualizaciones automáticas
  * Integración con sistema operativo

b) Aplicación Móvil
- Framework: Angular 15 + Ionic
- Características:
  * Diseño responsive
  * PWA capabilities
  * Acceso a características nativas del dispositivo

2.4 Capa de Comunicación
- GraphQL (Apollo)
- Características:
  * Queries optimizadas
  * Subscriptions para datos en tiempo real
  * Cache management
  * Error handling

3. MÓDULOS DEL SISTEMA
---------------------

3.1 Módulo de Administración (Schema: administrativo)
- Descripción: Gestión de autorizaciones y marcaciones del sistema
- Tipos de datos (ENUM):
  * estado_autorizacion:
    - EN ESPERA
    - CANCELADO
    - AUTORIZADO
    - NO_AUTORIZADO
  * tipo_autorizacion:
    - MARCACION
  * tipo_marcacion:
    - ENTRADA
    - SALIDA

3.2 Módulo de Configuración (Schema: configuraciones)
- Descripción: Gestión de configuraciones del sistema y dispositivos
- Tablas:
  * inicio_sesion:
    - id: bigint
    - usuario_id: bigint
    - id_dispositivo: text
    - hora_inicio: timestamp
    - hora_fin: timestamp
    - creado_en: timestamp
    - sucursal_id: bigint
    - token: text
    - tipo_dispositivo: tipo_dispositivo
  * local:
    - id: bigint
    - sucursal_id: bigint
    - creado_en: timestamp
    - usuario_id: bigint
    - equipo_id: bigint
    - is_servidor: boolean
  * rabbitmq_msg:
    - id: bigint
    - tipo_accion: text
    - tipo_entidad: text
    - entidad: text
    - id_sucursal_origen: bigint
    - data: jsonb
    - recibido_en_servidor: boolean
    - recibido_en_filial: boolean
    - exchange: text
    - key: text
    - class_type: text
- Tipos de datos (ENUM):
  * nivel_actualizacion:
    - CRITICO
    - MODERADO
    - MANTENIMIENTO
  * nivel_error:
    - INFO
    - ALERTA
    - PELIGRO
    - CRITICO
  * tipo_actualizacion:
    - MOBILE
    - DESKTOP
    - SERVIDOR_FILIAL
    - SERVIDOR_CENTRAL
  * tipo_dispositivo:
    - ANDROID
    - IOS
    - DESKTOP_WIN
    - DESKTOP_LIN
    - DESKTOP_MAC
    - WEBWEB_MOBILE
  * tipo_error:
    - APLICACION
    - BASE_DE_DATOS
