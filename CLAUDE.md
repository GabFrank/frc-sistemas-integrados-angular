# FRC Desktop App

## Stack
- Angular 15
- Electron 22
- TypeScript
- SASS
- Playwright (e2e)
- ESLint

## Build & Run
- Start: `npm start` (runs Electron + Angular in parallel)
- Build: `npm run build`
- Build prod: `npm run build:prod`
- Lint: `ng lint`

## Project Structure
- `src/app/` - Angular application source
- `app/` - Electron main process
- `e2e/` - End-to-end tests

## Conventions
- Follow existing code patterns and naming conventions
- Use Spanish for business domain terms as established in the codebase
